# Basepoint

Basepoint is a modern booking and business management platform for service-based businesses. It helps business owners manage services, clients, business hours, booking rules, payments, and recurring subscriptions in one clean dashboard.

## Features

- Business dashboard with organized admin tabs.
- Service management for one-time offerings.
- Client list with booking history.
- Business profile and availability settings.
- Booking rules for scheduling control.
- Payments section for payouts and payment methods.
- Subscriptions and memberships for recurring plans.
- Clean responsive UI built for fast management.

## Subscriptions

Basepoint supports recurring memberships and subscription-style plans for businesses like barbershops, salons, gyms, and studios.

Examples:
- $30/month for 5 haircuts.
- $300/month for unlimited haircuts.
- Yearly VIP membership plans.

## Tech Stack

- React
- TypeScript
- Tailwind CSS
- Dashboard-style component architecture
- Stripe-ready payment structure
